# Links for UI/UX Resources

Here are all the website links that was used as resources for inspiration or for actual implementation. More will be added until futher notice.
Below are the general descriptions from their respective pages for a short overview.

## Figma plugins

- [SimpleFlow](https://www.figma.com/community/plugin/751821593330638172/Simpleflow) (Effortlessly create user flow diagram inside Figma file with native FigJam connector and magically stay connected as you update your design!)
- [Shaders](https://www.figma.com/community/plugin/1081769622242809971/Shader) (Create up to 255 shades of your selected base color/s. That's right. Create shades of multiple selected colors all in 1 go!)
- [Contrast](https://www.figma.com/community/plugin/748533339900865323/Contrast) (Contrast makes it easy to check the contrast ratios of colors as you work. Select a layer and Contrast will immediately look for the color directly behind
  your selection and serve up the contrast ratio along with passing and failing levels from the Web Content Accessibility Guidelines (WCAG).
  If Contrast cannot find the background color, add another layer to your selection, and Contrast will compare them both.)
- [Font Awesome Icons](https://www.figma.com/community/plugin/774202616885508874/Font-Awesome-Icons) (This is the unofficial plugin for the Font Awesome Icon Collection. The plugin provides access to the entire free icon collection.
  All icons are in vector format. Plugin allows you to filter the list of icons by name.)
- [Ink Wireframe](https://www.figma.com/community/plugin/1071077298865016770/Ink-Wireframe) (Ink Wireframe plugin lets you create lo-fi wireframes.)
- [Noun Project](https://www.figma.com/@nounproject) (Noun Project is building a global visual language that unites us.
  Browse millions of icons and photos for anything at nounproject.com.)
- [ShadeCN](https://www.figma.com/community/file/1203061493325953101) (Beautifully designed components built with Radix UI and Tailwind CSS.)
